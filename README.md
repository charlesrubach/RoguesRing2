# Rogue's Ring 2

A modern reimagining of the classic Pebble app **Rogue's Ring**, rebuilt for the **Pebble Time 2** and **Pebble Round 2** using **JavaScript (Alloy)**.

## Overview

Rogue's Ring 2 brings back the minimalist, fast, and glanceable gameplay of the original Pebble version. Designed for next‑generation Pebble hardware, this version focuses on smooth animations, improved UI clarity, and a modular Alloy‑based code structure.

## Features

- Only compatible for **Pebble Time 2** and **Pebble Round 2**
- Built with **Alloy JS** for maintainability and modularity
- Clean, responsive UI adapted to round and rectangular displays
- QR Codes are now generated to watch the original video if you need a little more information
- Glyphs are now rendered using vectors instead of static pixelated images
